# 🐍 Python Practice

A collection of small Python scripts written while practicing core programming concepts — strings, file/OS handling, exception handling, and third-party libraries.

---

## 📌 About This Repository

This repo holds standalone practice problems solved while learning Python — each one focused on a specific concept rather than a full project. It's a working log of fundamentals being reinforced through code.

- 🎓 B.Tech CSE (AI/ML Specialization), Future University
- 🛠️ Focus: Core Python concepts — strings, OS module, exception handling, libraries
- 📈 Goal: Build strong fundamentals through consistent practice

---

## 🗂️ Problems

| File | Description | Concepts |
|------|-------------|----------|
| [problem1.py](./problem1.py) | Prints a multi-line poem to the console. | Multi-line strings, `print()` |
| [problem3.py](./problem3.py) | Converts text to speech using the `pyttsx3` library. | Text-to-speech, third-party libraries |
| [problem4.py](./problem4.py) | Lists the contents of a directory, handling missing-path and permission errors gracefully. | `os` module, exception handling |

---

## 🚀 Getting Started

```bash
git clone https://github.com/deepanshu-bisht-dev/python_practice.git
cd python_practice
python problem1.py
```

Some scripts need extra packages:

```bash
pip install pyttsx3   # for problem3.py
```
